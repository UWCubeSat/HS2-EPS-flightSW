# HSL-EPS-BatteryManager

This is the repository for battery manager using BQ25756. Any works for battery management could be stored under this repositoory.

# CMake
Current CMake will check main errors, so please use this feature to find the main errors in the code in library!

To run CMake, please run this command in HSL-EPS-BatteryManager directory

This configures the build

`cmake -S . -B build`

This build the project

`cmake --build build`
